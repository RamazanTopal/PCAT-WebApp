# PCAT Web Application with Node Js
A beautifully designed Photo Blog app with CRUD functionality

## CRUD functionality
- Create photo blog
- Read photo blog
- Update blog
- Delete blog


## Installation
    ```
    git clone https://github.com/RamazanTopal/PCAT-WebApp.git
    cd PCAT-WebApp
    ```
## Start
    ```
    npm start
    ```
